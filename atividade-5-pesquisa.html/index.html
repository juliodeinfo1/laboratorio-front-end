Formatação de texto com <b> - HTML
Júlio césar INFO1

fonte: https://www.w3schools.com/html/html_formatting.asp

Na linguagem de HTML, o codigo <b> é usado para NEGRITAR qualquer palavra.
A função do negrito em grande maioria das vezes é dar destaque as palavras para
que "chamem atenção" e facilitem a leitura de quem está olhando a nossa página
em busca de informações expecificas.
